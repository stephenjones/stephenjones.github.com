Git Commands

A list where I can find them when out and about... (until they are committed to memory)

  • git remote add [name] [remote location]: Add a brief name for a remote location to reduce the typing
  • git push [name] [branch]: Push to GitHub / remote repository
  • git pull [name] [branch]: Bring down and merge the latest from GitHub / remote repository
  • git branch -r: List the remote branches
  • git checkout [other_branch]: Move to the other branch and create a repository with the status of its HEAD
  • git checkout -b [new_branch] [old_branch]: Create a new branch from one that already exists, checkout and move there
  • git reset --hard HEAD^: Undo the last commit in your current branch
  • git revert HEAD *: Revert the files 'added' so they can be ignored or readded once moved
  • git branch [newname] [existing branch]: Create a separately tracked repository
  • git merge [other_branch]: Include the other branch's details into this branch (often the 'master')
